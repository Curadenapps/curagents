## How to Use This File

The `academy-routing` procedure filters entries by:
1. `role`: `clinician` or `patient`
2. `topic`: keyword match against tags
3. `level`: `beginner | intermediate | advanced` (optional)

Return top 1–3 matches. Always prioritise `clinician` entries for clinical topics
regardless of the role input.
